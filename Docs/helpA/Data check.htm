Data check

Rami program allows you to check if the entered data is correct. Program checks correctness of the connection between the components of the method and consistent with the data model. Using the control mechanism (“Check” option) helps to determine the importance of model components and size of the model. if the program finds an error, show an error message and inform about the type and position of the incorrect component. If the data and relationships are correct, the consistency check ends with a message: “The data is consistent”. If an error occurs, it has no consequences, it means that the data model needs to be adjusted before agreeing the risk parameters for consistent data. The program function controls the assignment:

ü  assets to regulation,

ü  assets to security properties,

ü  attack to assets,

ü  attack to threat,

ü  attack to attacker.
