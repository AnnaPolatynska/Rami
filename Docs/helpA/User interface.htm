User Interface

Rami offers an easy-to-use interface. It includes a group of buttons in the top menu to managed data registers and control their integrity. It incluse also direct help functions and information about Rami program. Input data and risk calculation is in three main tabs:

Instrument - contains the name and description describing the characteristics of the measuring device.

Dictionaries - a group of tabs covering: legal regulations, assets, security properties, threats, attacks and attackers.

Relationships - to build a coherent data model. The model describes the relationship between: regulation - assets, assets -  security properties, attack - assets, attack - threat, attack – attacker and attack - risk.
