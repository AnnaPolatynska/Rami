Program features

The main goal of creating the program was to build a tool that supports analysts in determining indicators that are important for the safety of measuring instruments in a systematic and formalized way.

The program offers:

ü  reliable risk assessment,

ü  verification of the correctness and consistency of data,

ü  interface that guides navigation and changes of model,

ü  text and graphical interpretation of input parameters and result,

ü  possibility to save data and results in an XML file.

The program enables quick analysis of threats for 10 attempts.
