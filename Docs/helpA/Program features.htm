Program features

The main goal of creating the program was to build a tool that in a systematic and formalized way supports analysts in developing reliable indicators relevant to the safety of measurements of all instruments. The program offers:

ü  reference to a credible formalism of risk assessment,

ü  verification of the correctness and consistency of data,

ü  a friendly interface that navigates and changes model parameters,

ü  text and graphic interpretation of input parameters and calculation result,

ü  registration of data and calculation results in an XML file.

Due to the specificity of the issue, the program offers a quick risk analysis for the trials, which can be called subsequent approximations of the target risk. I am talking about a specific interaction between the expert and the program, when the calculated risk may affect the adjustment of parameters asked by the expert in order to get a better approximation of the analysis result.
